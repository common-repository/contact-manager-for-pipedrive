== Description ==

Contact Manager for Pipedrive lets you automatically create deals in Pipedrive from any WPForms form.  Under the form settings, just map form fields to Pipedrive fields, so any form submission will be sent to Pipedrive.

####Privacy Policy & Terms of Service

I don't use or even receive any data from this plugin, and you can use it however you want.

== Installation ==

####Option 1:

- Go to the Add New plugins screen in your WordPress admin area.
- Using the search box, find the Contact Manager for Pipedrive plugin and click the install button and then activate the plugin.

== Usage Instructions ==

You will need a Pipedrive API Key which can be found on the Pipedrive dashboard under Personal Preferences -> API.

Once you've downloaded and installed this plugin on your site, go to the Settings -> Contacts for Pipedrive submenu to set your Pipedrive API Key.

If you want to add deals to Pipedrive from a WPForms form, open the WPForms form and click on Settings -> Pipedrive to choose your preferences.

== Actions and Filters ==

We don't have any filters or actions at this time, but please feel free to request any you want!

== Frequently Asked Questions ==

= How do I use this thing? =

Check the Usage Instructions above, but basically set the Pipedrive API Key in Settings -> Contacts for Pipedrive and then set fields in WPForms form Settings -> Pipedrive.
